
# Flutter Authentication İşlemleri

Bu proje uygulamalarda kullanılan firebase işlemlerinin bir dökümantasyonu niteliğindedir.


## Özellikler

- Email/Şifre Kayıt
- Email/Şifre Girişi
- Oturumu Kapat
- Kullanıcıyı Sil
- Parola Değiştir
- Email Değiştir
- Google ile Giriş
- Tel No Giriş
